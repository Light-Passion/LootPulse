# Goals Manifest

This manifest documents the core goals of the LootPulse project.

## Active Goals

- [G-001] **Project Setup**: Initialize the WPF desktop shell on .NET 9.0 and configure the GOTCHA framework structure.
- [G-002] **PoE2 Subject Matter Validation**: Integrate constraints from `poe2_validator` (uncut gems, gold untradability, currency valuation standards).
- [G-003] **Transparent Overlay**: Build a premium, mouse-penetrable (click-through) transparent overlay window for monitoring item values in-game.
- [G-004] **Build File Syncing**: Parse the player's PoE2 `.build` file to highlight leveling gear dynamically.
- [G-005] **Market API Integration**: Monitor PoE Trade and poe.ninja APIs for real-time economy updates.
